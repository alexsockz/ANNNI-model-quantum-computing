# Quantum Computing Project on the ANNNI Model
Questo progetto è stato svolto da [Alessio Valle](https://github.com/alexsockz), [Manuele Nardone](https://github.com/manuelenard-eng) e [Marco Fava](https://github.com/manuelenard-eng) come progetto per l'esame di Computazione Quantistica (tenuto dal professor [Michele Grossi](https://michele-grossi.web.cern.ch/) e [Bassi Angelo](https://www.units.it/persone/index.php/from/abook/persona/8253) presso l'università di Trieste).  
Lo scopo del progetto è stato di studiare ed emulare andando oltre il lavoro pubblicato nel paper [Quantum phase detection generalisation from marginal quantum neural network models](https://arxiv.org/abs/2208.08748).  
Più di preciso, siamo partiti con l'aiuto di due tutorial ([Supervised and unsupervised quantum machine learning models for the phase detection of the ANNNI spin model](https://pennylane.ai/qml/demos/tutorial_annni) e [Seeing quantum phase transitions with quantum computers](https://pennylane.ai/qml/demos/tutorial_quantum_phase_transitions)) ed abbiamo sviluppato una nostra [VQE](src/vqe_and_search/), introdotto uno studio con o senza rumore per un circuito QCNN e QAD per il modello ANNNI (vedi [qua](classification_and_unsupervised)), ed infine abbiamo studiato il comportamento di diversi osservabili all'applicazione di una evoluzione temporale ([time evolution](src/ANNNI-TimeEv.ipynb) e [classification](src/Classif_Time_Evol.ipynb))
